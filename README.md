# whistle.view-md5


该插件是在 Network / Inspectors 自定义 Tab 的示例（whistle >= `v2.9.0`），通过在该自定义的 Tab 可以查看北斗上报的Base64的Json解析数据。



**安装whistle**

```

yarn global add whistle 

npm install -g whistle

```



**whistle 插件使用**

```



# 安装

w2 i whistle.hz-parse-base64  

# 卸载
w2 uninstall whistle.hz-parse-base64  



```
